# 学习 Skill：对话式学习指南

## 目标

通过对话方式创建一个学习 skill，用于引导学习者理解项目结构、学习方法和实践步骤。

## 对话示例

### 学生：你能帮我了解这个项目吗？

助教：当然可以！这个仓库名为 `hhfyeg`，目前包含一个根目录 `README.md` 和一个子目录 `hhfyeg`，子目录中也有一个 `README.md`。我们可以把这个项目构建成一个学习技能，用于学习项目说明、功能设计和提交流程。

### 学生：我应该从哪里开始？

助教：先打开根目录下的 `README.md`，补充项目简介、核心功能和使用说明。然后建立一个学习 skill 文档，例如 `hhfyeg/learning-skill.md`，把学习过程整理成对话形式。

### 学生：这个学习 skill 应该包含哪些内容？

助教：建议包含以下内容：
- 项目简介和学习目标
- 主要任务步骤
- 核心知识点讲解
- 实践建议和扩展思路
- 对话式问答示例，帮助学习者主动思考

### 学生：完成后我该怎么提交？

助教：你可以使用 `auto-git.sh` 脚本自动执行 `git add`、`git commit` 和 `git push`，然后在 GitHub 仓库中查看提交记录。

## 实践步骤

1. 阅读并完善项目 `README.md`。
2. 在 `hhfyeg/learning-skill.md` 中创建对话式学习内容。
3. 使用 `auto-git.sh` 提交并推送到 GitHub。
4. 使用 `/export` 导出本次对话内容，作为作业提交证明。

## 说明

这个 `learning-skill.md` 文件是一个学习技能示例，展示了一种通过对话引导学习的方式。你可以根据教师演示过程进一步扩展内容，例如添加项目功能细节、代码实现说明和实战练习。
